# About the Diagrams

These diagrams were all created using dbt_dotdiag.py with the exception of fig1_dbt_lin_graph_exmple.png which is a screen capture of model lineage diagram generated by a dbt document server.

Each diagram corresponds to a figure presented in the README.md file. The commands used to generate them are contained in the file example/scripts/dbt_dotdiag_script.md.
