
The following Linux/Mac commands were used to create the diagrams used in the README.md file. 

NOTE: Figures 1 was captured directly from the standard dbt documentation portal's web interface, not generated by the tool dbt_dotdiag.

``` shell
# environment variable to simplify the scripts
# 
DEV_DIR="$HOME/main/proj/dev/dbt"
CODE_DIR="$DEV_DIR/code"
OUT_DIR="$DEV_DIR/output"
MANIFEST_PATH="$DEV_DIR/dbt_targets/anon_manifest.json"
```

**Figure 2: The dbt_dotdiag equivalent of Figure 1**

NOTE: The diagram shown in Figure 2 of README.md is a cropped version of the diagram that is generated with the script below. The diagram was cropped to produce a figure that is comparable Figure 1 with respect to the total number of models and complexity and depth of lineage.

``` 
python $CODE_DIR/dbt_dotdiag.py \
--target-models fct_email_campaign_user_engagement,rpt_user_email_campaign_results,rpt_email_campaign_impact \
--parent-depth 2  \
--output-path $OUT_DIR/fig2_equivalent_to_fig1 \
--pdf --svg \
--manifest-path $MANIFEST_PATH
```
